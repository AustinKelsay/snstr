# NIP-10 Tests

This directory contains tests for the NIP‑10 thread utility functions.

## Components Tested

- Parsing of marked and positional `e` tags
- Parsing of `q` tags
- Creation helpers for reply and quote tags

## Running the Tests

```bash
npm run test:nip10
```

## Test Files

- `nip10.test.ts`: Unit tests for the NIP‑10 utilities
